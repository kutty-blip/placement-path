# 🚀 Smart Placement Preparation & Guidance System

🔗 Live Demo: https://smart-placement-prep-system.netlify.app

---

## 📌 Overview

The **Smart Placement Preparation & Guidance System** is a web-based platform designed to help students prepare effectively for technical placements.

It provides:
- Company-specific interview preparation
- Frequently asked DSA concepts
- AI-based guidance for structured learning

---

## 🎯 Features

### 🔹 1. Company + Concept Based Questions
- Select a **company** and a **DSA concept**
- Get **previously asked interview questions**
- Helps in targeted preparation

---

### 🔹 2. Company-wise Important Concepts
- Select a **company**
- View **most frequently asked DSA topics**
- Focus on high-priority concepts

---

### 🔹 3. Interactive Dashboard
- Clean and responsive UI
- Easy navigation for users
- Built for better user experience
